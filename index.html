02 🤖 AGENTIC AI
Supplier Email Extractor - Autonomous Email Processing Agent
↗
⏱ Eliminates 1–2 hours/day of manual data entry · ~€5/month running cost · 30–60 emails/min throughput

A fully autonomous AI agent that converts unstructured supplier emails into structured business data - no human intervention required. Targets a universal SMB pain point: manually copying order details from supplier emails into spreadsheets. Once configured, the agent runs in the background, picks up new labeled emails via IMAP, extracts order data using GPT-4o with Pydantic schema validation, and appends rows to a live Google Sheet dashboard - flagging low-confidence extractions for human review.

Key capabilities: bilingual extraction (Italian + English), PDF processing with text and vision fallback for scanned documents, confidence scoring, ambiguity handling, multiple-order detection, deduplication via Gmail labels, per-call token cost logging, and a provider-agnostic LLM interface (OpenAI, Claude, Gemini swappable). Deployed as a standalone Python project on client machines - no shared infrastructure, setup fee model.

Target users: restaurants, hotels, bakeries, retailers, construction firms, beauty salons, bookkeepers - any SMB receiving 10+ supplier emails/day.
